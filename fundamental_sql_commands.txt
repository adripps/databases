1. INSERT INTO adds data, UPDATE updates data, and DELETE and DROP TABLE deletes data.

2. CREATE:
	Initializes a table and creates columns and establishes their data type

	CREATE TABLE <table name>(			
	   <column 1 name> <data type for column 1>,
	   <column 2 name> <data type for column 2>
	);

    INSERT INTO: 
	Inserts data into a given table

	INSERT INTO <table name> (<column 1 name>, <column 2 name>) //
	VALUES
	(<insert first piece of data into column 1>, <insert first piece of data into column 2>),
	(<insert second piece of data into column 1>, <insert second piece of into data column 2>);

    UPDATE: 
	Updates existing data within a table

	UPDATE <table name> SET <column> = <updated data>, 
	WHERE <unique identifier>

    DELETE:
	Deletes data from a table

	DELETE FROM <table name>
	WHERE <unique identifier>

    ALTER TABLE:
	Adds, deletes, or updates columns of a given table	

	ALTER TABLE <table name>
	<ADD or DELETE or UPDATE> COLUMN <column name> <data type>;
	
    DROP TABLE:
	Deletes an entire table when given the table’s particular name

	DROP TABLE <table name>;

3. Text can be used for names or descriptions which can offer a variety of real world uses including keeping track of item names and a short description of said item. Boolean is used for data that can either be true or false and could be useful for things like keeping track of whether or not a a user has paid for a premium feature. Numeric is a data type that holds numeric values and can be useful for storing identification numbers.
